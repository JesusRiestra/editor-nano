## NANO

Para iniciar Nano utilizamos el comando --> **nano nombredelarchivo** (si el archivo indicado no existe, se crearÃ¡).

Guardar archivo --> **Ctrl + O**

Salir del editor --> **Ctrl + X**

Eliminar linea completa --> **Ctrl + K**

Seleccionar texto --> **Alt + A**

Copiar texto --> **Alt + 6**

Pegar copiado --> **Ctrl + U**
